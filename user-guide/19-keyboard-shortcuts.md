# Keyboard shortcuts

*The complete shortcut reference.*

On Windows and Linux, use `Ctrl` wherever this guide shows `‚åò`. Press `‚åò/` at any time to open the in-app shortcuts overlay.

## Requests

| Shortcut | Action |
|---|---|
| `‚åò‚Üµ` | Send the current request |
| `‚åòT` | Open a new empty request tab |
| `‚åòW` | Close the active tab |
| `‚åòS` | Save the request (in place if already saved, else the Save dialog) |

## Navigation

| Shortcut | Action |
|---|---|
| `‚åòK` | Focus the Omnibox in command mode (the command palette) |
| `‚åòP` | Focus the Omnibox in search mode (quick-open a saved request or history entry) |
| `‚åò\` | Cycle Workbench focus: both panes ‚Üí response ‚Üí request ‚Üí both |
| `‚åòE` | Toggle the Environments panel (switch environment) |
| `‚åò1` | Toggle the **Collections** panel |
| `‚åò2` | Toggle the **History** panel |
| `‚åò3` | Toggle the **Environments** panel |
| `‚åò4` | Toggle the **Flows** panel |
| `‚åò5` | Toggle the **Connections** panel |
| `‚åò6` | Open **Settings** |
| `‚åò/` | Toggle the keyboard-shortcuts overlay |

## Editor

| Shortcut | Action |
|---|---|
| `‚åòF` | Find in the response |
| `Esc` | Close the shortcuts overlay, or close a floating (unpinned) panel |

## The Status strip hints

The bottom Status strip always shows a compact hint row: `‚åò‚Üµ Send`, `‚åòK Commands`, `‚åòP Open`, `‚åò\ Focus`, `‚åòE Env`, `‚åò? Keys`.
